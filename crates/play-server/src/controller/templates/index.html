BTC Transfer

Build unsigned Taproot PSBTs with UTXO control, then validate and broadcast signed transactions.

How to use

  1. Enter the from/recipient/amount; the network auto-detects from the sender address. Set fee rate and choose UTXOs or change address.
  2. Create the unsigned PSBT and move it to the offline signer via QR or copy.
  3. Paste or scan signed hex, validate, and broadcast.

BTC Signer

Offline PSBT signer that stores keys locally, reviews details, and exports signed hex and QR. Install as a PWA for a dedicated offline signer.

How to use

  1. Install as a PWA (Add to Home Screen) and keep the signer offline.
  2. Add a key pair for the input address (WIF or hex) and disconnect from the internet.
  3. Scan or paste the unsigned PSBT and review the decoded summary.
  4. Sign and share the signed hex via QR or copy back to BTC Transfer.
Taproot key lab Open tool

BTC Keys

Taproot key generation and export for mainnet/testnet with deterministic and encrypted options.

How to use

  1. Select mainnet/testnet, generate or derive a key, and confirm the Taproot address.
  2. Copy the private key (raw or WIF) or show QR; use BIP38 for encrypted backups.
  3. Use the address in BTC Transfer while keeping the private key offline.
Security posture: Keep private keys offline, prefer encrypted QR when sharing or printing, and verify recipient address, fee rate, and network before broadcasting. Offline-ready: Each tool is a single all-in-one HTML page, so you can download it and run locally.
